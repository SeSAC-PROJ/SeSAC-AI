## 📂 About Our Project  

> KSEB-AI Repository는 **KSEB 4기 2팀의 AI Algorithm & Backend 코드**를 담고 있습니다.  
> 다양한 실험과 모델 개발, 그리고 API까지 **AI 개발의 모든 여정을 기록**합니다.

📌 주요 기능:
- 🤖 머신러닝/딥러닝 모델 & 관련 알고리즘 학습 및 평가
- ⚡ FastAPI 기반의 백엔드 서비스
- ☁️ 발표의 핵심인 시선, 자세, 표정, 음성에 대한 분석 및 종합적인 평가 기능
